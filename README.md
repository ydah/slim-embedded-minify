# SlimEmbeddedMinify

A slim file to minify embedded code.

## Installation

Just install the slim-embedded-minify gem

```
gem install slim-embedded-minify
```

or if you use bundler put this in your `Gemfile`

```
gem 'slim-embedded-minify'
```
